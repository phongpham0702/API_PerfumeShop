## Configuration Steps


Use 2 terminal/cmd/powershell window

In 1st window
```
1) install server modules
$ cd ./server
$ npm install

2) Run server
$ npm start
```
In 2nd window
```
3) install client modules
$ cd ./client
$ npm install

4) Run client
$ npm dev
```
